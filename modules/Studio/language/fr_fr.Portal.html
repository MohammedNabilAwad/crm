Vous pouvez choisir les options suivantes pour modifier votre portail SuiteCRM:

  • Modifier un module. Choisissez cette option pour modifier le module de Suivi de bug, de Tickets ou de Leads.
  • Télécharger une feuille de Style. Sélectionnez cette option pour modifier la feuille de style du portail SuiteCRM par défaut.
  • Portail Sync. Sélectionnez cette option pour synchroniser le portail SuiteCRM avec le serveur SuiteCRM.

Pour modifier un module, procédez comme suit :

  1. Sélectionnez le module que vous souhaitez reconfigurer dans le portail SuiteCRM.
  2. Sélectionnez la vue que vous voulez modifier.
    Le portail éditeur affiche une barre d’outils sur la gauche et les champs de vue de détail sur la droite. Pour modifier pour modifier l’affichage ou affichage de la liste, sélectionnez le lien approprié ci-dessus. La barre d’outils affiche les champs disponibles qui ne sont actuellement pas dans la vue de détail. Il affiche également un champ vide et un champ vide. Utilisez un champ vide pour créer des espaces dans une vue. Par exemple, utilisez un champ vide à côté de la Description pour créer un espace suffisant pour entrer dans une description détaillée. Utilisez un champ vide comme espace réservé que vous pouvez remplacer par un nom de champ dans le futur.
  3. Pour ajouter un champ de la table de la barre d’outils à la vue de détail, cliquez sur Ajouter une ligne. Sélectionnez le champ dans la barre d’outils et faites-le glisser vers le champ vide.
  4. Pour supprimer un champ de la vue, sélectionnez-le et faites-le glisser vers la barre d'outils.
  5. Cliquez sur Enregistrer pour enregistrer vos modifications.

Pour télécharger une feuille de style, procédez comme suit :

  1. Sélectionnez l’onglet correspondant pour les affaires, Bug Tracker ou les Newsletters.
  2. Cliquez sur Parcourir, naviguez jusqu'à l’emplacement de la feuille de style, puis sélectionnez-la.
  3. Cliquez sur Télécharger pour afficher la nouvelle feuille de style.

Pour synchroniser le portail SuiteCRM sur le serveur SuiteCRM, procédez comme suit :

  1. Entrez l’URL de l’instance de portail que vous souhaitez mettre à jour et cliquez sur Go.
  2. Saisissez vos nom d'utilisateur et mot de passe et cliquez Démarrer synchronisation.
